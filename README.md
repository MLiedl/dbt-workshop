# DBT Data Analysis Workshop

---
### What is Data Analysis?

Data analysis is about turning raw information into meaningful insights to help people 

- Make better decisions (e.g., businesses, governments)
- Understand trends (e.g., weather, social media)
- Solve problems (e.g., why sales dropped)

This requires inspecting, cleansing, transforming, and modeling the data into useful information and for this we will need the following tools:

- Python required by the other tools
- Jupyter as Jupyter Notebooks are used as the interactive teaching platform
- a Text Editor  for writing bash or powershell scripts
- DuckDB an SQL like Database System for easily manipulating the formatted data
- DBT Tool a command-line tool for transforming and modeling data in a SQL Database System
- Seaborn a Python data visualization library for informative statistical graphics
- Streamlit for easy building of interactive data apps in Python

---

First, we need to ensure that Python, Jupyter and a Text Editor are available. These will be used in preparing a working DBT environment, which then can be used for performing a full Data Analysis process on a chosen dataset. These steps are explained in the following three documents:

- 1 - Installing Python, Jupyter and a Text Editor (text file)
- 2 - From Zero to a Working DBT (Jupyter Notebook)

# 🚀 DBT Data Analysis Workshop

Welcome! 👋  
In this workshop, you’ll go from **zero setup → to building a real data project** using modern tools used by data analysts.

---

## 📊 What is Data Analysis?

Data analysis is about turning **raw data** into **useful insights**.

Think of it like this:
- 📉 Why did sales drop last month?
- 🌦️ What patterns are there in weather data?
- 📱 What are people doing on social media?

To answer questions like these, we:
- 🔍 Inspect data  
- 🧹 Clean it  
- 🔄 Transform it  
- 📈 Visualise and model it  

---

## 🧰 Tools You’ll Use

Don’t worry if these are new — we’ll set everything up step by step.

- 🐍 **Python** – the main programming language (everything else depends on it)
- 📓 **Jupyter Notebook** – interactive environment where you’ll run code and see results instantly
- ✏️ **Text Editor** – for editing scripts (e.g., VS Code or similar)
- 🗄️ **DuckDB** – a lightweight database you can run locally (no setup drama)
- 🔧 **DBT (Data Build Tool)** – transforms raw data into clean, structured data using SQL
- 📊 **Seaborn** – for creating clear and informative charts
- 🌐 **Streamlit** – for turning your analysis into a simple interactive app
