# 1. Installing Python, Jupyter and a Text Editor
## 🚀 Let’s Get You Set Up for Data Analysis
  
  Before we start building cool data projects, we need to install a few essential tools:  
- **Python** → the programming language everything else runs on
- **Jupyter Notebook** → where you’ll write and run your code interactively
- **A Text Editor** → for writing simple scripts and configs
  
Don’t worry — we’ll keep this **simple, lightweight, and beginner-friendly**.  
  
---
# 🪟 Windows Setup
## Step 1: Install Python (the easy way)
- Go to: [https://www.python.org/downloads/](https://www.python.org/downloads/)
- Download the latest version
- Run the installer
  ✅ IMPORTANT: Tick **“Add Python to PATH”** before clicking install  
## Step 2: Install Jupyter
  
  Open **Command Prompt** and type:  
  
  ```
  pip install notebook
  ```
  
  To launch Jupyter:  
  
  ```
  jupyter notebook
  ```
  
Your browser will open — this is your coding workspace 🎉  
## Step 3: Install a Text Editor
  
  Simple options:  
- **Notepad++** (lightweight, great for beginners)
- **VS Code** (more powerful, still beginner-friendly)
  
  👉 Recommendation: Start with **VS Code**  
  
---
# 🐧 Linux Setup (Ubuntu / Debian-based)
## Step 1: Install Python and pip
  
  Open Terminal:  
  
  ```
  sudo apt update
  sudo apt install python3 python3-pip
  ```
## Step 2: Install Jupyter
  
  ```
  pip3 install notebook
  ```
  
  Run it:  
  
  ```
  jupyter notebook
  ```
## Step 3: Install a Text Editor
  
  Options:  
- **nano** (already installed, super simple)
- **gedit** (clean graphical editor)
- **VS Code** (if you want something more advanced)
  
Install gedit:  
  
  ```
  sudo apt install gedit
  ```
  
---
# 💻 Chromebook Setup (Linux Partition)
  
  If you haven’t already:  
  
  👉 Enable **Linux (Developer Environment)** in settings  
## Step 1: Open the Linux Terminal
## Step 2: Install Python
  
  ```
  sudo apt update
  sudo apt install python3 python3-pip
  ```
## Step 3: Install Jupyter
  
  ```
  sudo apt install python3-notebook
  ```
  This installs Jupyter Notebook system-wide. Now you can run it:  
  
  ```
  jupyter notebook
  ```

note: If your notebook is opened and blocks your terminal, you can use Ctrl-Z to exit the application.


## Step 4: Install a Text Editor
  
  Simple options:  
  
  ```
  sudo apt install nano
  ```
  
  Or for a better experience:  
  
  ```
  sudo apt install gedit
  ```
  
---
# 🧠 What You Should Have Now
  
  By this point, you should be able to:  
- Run `python` or `python3` in a terminal
- Start Jupyter with `jupyter notebook`
- Open and edit files in a text editor
