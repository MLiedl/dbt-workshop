## 🧭 Suggested Study Roadmap — From Beginner to Professional Analyst

This workshop introduced many modern analytical tools and ideas.

Below is a suggested progression path for developing professional-level proficiency.

---

### 🌱 Stage 1 — Foundations

Focus:  learn the core analytical building blocks.

#### Topics
- Python basics
- basic SQL
- spreadsheets and tabular thinking

#### Recommended Tools
- Python
- Jupyter Notebook
- DuckDB
- Pandas

---

### 🧱 Stage 2 — Data Transformation and Modelling

Focus:  learn how professional analytical pipelines are structured.

#### Topics
- SQL modelling
- staging vs marts
- analytical thinking
- reusable transformations
- data lineage
- data quality testing

#### Recommended Tools
- DBT
- Jinja-templated SQL
- DuckDB
- Git/GitHub

---

### 📊 Stage 3 — Visualisation and Communication

Focus:  learn to explain findings visually and clearly.

#### Topics
- exploratory analysis
- storytelling with data
- chart design
- dashboards
- stakeholder reporting

#### Recommended Tools
- Matplotlib
- Seaborn
- Plotly
- Streamlit

---

## 🏗️ Stage 4 — Professional Analytics Engineering

Focus:  learn how modern data platforms operate in organisations.

#### Topics
- orchestration pipelines
- cloud data platforms
- CI/CD
- testing automation
- deployment
- version control workflows

#### Recommended Tools
- DBT Cloud
- GitHub Actions
- Airflow
- BigQuery / Snowflake / Databricks
- Docker

---

## 🧠 Stage 5 — Advanced Analytics and Data Science

Focus:  move from descriptive analytics into prediction and optimisation.

#### Topics
- statistics
- forecasting
- machine learning
- experimentation
- recommendation systems

#### Recommended Tools
- Scikit-learn
- XGBoost
- TensorFlow / PyTorch
- JupyterLab

---
